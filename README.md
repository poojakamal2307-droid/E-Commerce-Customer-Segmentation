# E-Commerce Customer Segmentation using Python & SQL

## Project Overview
This project performs customer segmentation using RFM (Recency, Frequency, Monetary) analysis on e-commerce transaction data.

The objective is to identify valuable customers and support targeted business decision-making.

---

## Tools & Technologies
- Python
- Pandas
- MySQL
- SQLAlchemy
- SQL

---

## Workflow
1. Transaction data stored in MySQL database
2. Data extracted using Python
3. Data preprocessing and feature engineering
4. RFM analysis performed
5. Customers segmented into business categories
6. Results stored back into SQL database

---

## Customer Segments
- Best Customers
- Loyal Customers
- Recent Customers
- At Risk Customers
